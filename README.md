# Tetris Game (C++ / CMake)

A classic Tetris desktop game developed in C++ using CMake.

## Features
- Line clearing logic
- Increasing speed with score
- Keyboard controls
- Smooth gameplay

## Controls
- Left / Right Arrow – Move
- Up Arrow – Rotate
- Down Arrow – Fast drop

## Build Instructions
```bash
mkdir build
cd build
cmake ..
cmake --build .
```

## Platform
- Windows (tested)
